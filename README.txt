Esta es la primera versión del programa RACIONAMIENTO de parte de Técnica.


---------INSTALAR---------

Si es la primera vez que descarga este programa, recuerde ejecutar el
instalador localizado en la carpeta "install" y esperar a que el programa
indique "tocar una tecla para continuar" y se cierre.

---------COMO UTILIZAR EL PROGRAMA---------

Para utilizar el programa, primero ingrese la carpeta con la lista de turno
actualizada de los tecnicos, luego elija la hoja de excel donde se encuentra
la lista de turno.

Una vez hecho lo anterior, con la opcion "Guarda" podra elegir en que carpeta
generar los partes diarios.

Si el programa no recibe bien los parámetros, existe un backup del mismo en la carpeta
"resourse", se recomienda hacer una copia de este, reemplazando el documento dañado, y
no trabajar directamente del backup.
